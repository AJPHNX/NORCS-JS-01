Employee Rewards

In this activity, we'll practice using if else conditional statements.


Northwest Marketing Services


Northwest Marketing Services, based in Seattle, WA, provides employee reward solutions to financial institutions. You are part of a team of developers working to create the system that manages those customers referrals and employee rewards.


You've been tasked with creating the logic that will determine the type of gift that is to be distributed to employees of First Liberty Bank who have met their incentives for the month of December.

Directions

Construct conditional if else statements using the following logic to determine if an employee is eligible for referral gifts:


  1. When the employee has 1 year with the company: If the employee has 1 new checking account, output: 'Ship $50.00 Amazon Gift Card', otherwise output: 'Incentives not met'
  2. When the employee has 2 years with the company: If the employee has 3+ new checking accounts, output: 'Ship $400.00 Amazon Gift Card', otherwise output: 'Ship $100.00 Visa Gift Card'
  3. When the employee has 3 years with the company: If the employee has 2+ new checking accounts, output: 'Ship $700.00 Visa Gift Card', otherwise output: 'Ship $350.00 Visa Gift Card'
  4. When the employee has 5+ years with the company: If the employee has 1+ new checking accounts, output: 'Ship $3500.00 Visa Gift Card', otherwise output: 'Ship set of knives'
